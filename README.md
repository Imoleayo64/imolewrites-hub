# 🎓 ImoleWrites Research Hub

**A Comprehensive AI-Powered Suite for Academic Excellence**

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://imolewrites.streamlit.app)

ImoleWrites Research Hub is a premium, multi-tool SaaS platform designed specifically for academic researchers, students, and scientific writers. It eliminates hours of manual formatting and literature hunting by combining advanced API integrations with large language models to streamline the manuscript drafting workflow.

## 🚀 Core Features

* **🤖 Smart-Citing Agent:** Paste raw manuscript drafts to instantly extract, correct, and format perfect APA parenthetical citations and full reference lists.
* **📚 Literature Sourcer:** Scour the global Crossref database for verified, high-impact peer-reviewed journals. Export complete bibliographies instantly in `.bib` (BibTeX) or `.txt` formats.
* **✍️ Style Converter:** Structurally convert existing reference lists between APA, Vancouver, and Turabian styles with total precision and strict formatting adherence.
* **🧠 Academic Humanizer:** Rewrite robotic, AI-generated text to bypass AI detectors (like Turnitin and GPTZero) while maintaining a rigorous, doctoral-level academic tone and preserving vital scientific terminology.

## 🛠️ Technology Stack

* **Frontend:** [Streamlit](https://streamlit.io/) (Python)
* **AI Engine:** [Groq API](https://groq.com/) (LLaMA-3 70B Versatile)
* **Academic Databases:** [Crossref API](https://www.crossref.org/services/metadata-retrieval/)
